# Particle Launcher · 粒子发射器系统 · 完整描述

---

## 一、核心系统架构

基于 **Electron + Vite + TypeScript** 构建的桌面级三维粒子交互平台。

- **渲染层**：Three.js WebGL 引擎
- **后处理管线**：EffectComposer + UnrealBloomPass + 自定义 finale Shader
- **色调映射**：ACES Filmic Tone Mapping，曝光值 0.85
- **粒子规模**：15,000 颗高性能粒子（BufferGeometry + ShaderMaterial）
- **输出画质**：温润柔和，无刺眼高光

## 三、粒子着色器效果（柔和版）

### 三层粒子结构
```
外光晕：exp(-d * 1.8)          — 大范围柔和扩散
中环：smoothstep(0.6, 0.4, d) - smoothstep(0.4, 0.15, d)  — 宽过渡带
柔核：exp(-d * 6.0)            — 中心柔和发光（非锐利）
```

### 合成参数
- **alpha 合成**：光晕 0.4 + 中环 0.35 + 核心 0.6
- **色彩保留**：核心仅微混白 35%，主体保留本体色
- **混合模式**：Additive Blending 加法混合，透明度柔和叠加

### 动画系统
- **多频呼吸**：低频 1.2Hz 主体起伏 + 高频 3.7Hz 微观颤动 + 8-10Hz 跳动抖动
- **脉冲尺寸**：sin(4t + 3φ) * 0.08
- **形态旋转**：整体绕 Y 轴持续旋转，角度累计

## 四、形态特定动态运动

由 **SHAPE_DYNAMICS** 配置表驱动，每个形态拥有独立四参数：

| 参数 | 说明 | 典型值范围 |
|------|------|-----------|
| rotationSpeed | 绕 Y 轴旋转角速度 | 0.03 ~ 0.80 rad/s |
| flowY | Y 方向流动位移量 | -2.0 ~ +0.5 |
| flowSpeed | 流动循环速度 | 0.2 ~ 2.0 |
| jitter | 跳动抖动强度 | 0.01 ~ 0.15 |

**循环包裹**：粒子掉出视口边界时自动回到对侧，防止视觉断裂。

## 五、手势与交互系统

### MediaPipe Hands 手势识别
- **支持手势**：pinch（捏合）、peace（胜利）、fist（握拳）、open_palm（张开手掌）、point（指）
- **触发机制**：手势匹配 → 1.5 秒冷却期 → 启动绑定软件
- **状态反馈**：状态栏实时显示手势识别结果

### 鼠标交互
| 操作 | 效果 |
|------|------|
| 悬停 | 粒子受径向排斥 + 切向涡旋力，产生流动感 |
| 点击空白处 | 200 粒子在点击位置 burst 炸开扩散 |
| 点击饰品 | Raycaster 射线检测，启动对应软件 |
| 右键拖拽 | 自由旋转相机视角 |
| 滚轮 | 缩放相机距离（限制 4-15 单位）|

### 动量物理
- **手部速度向量**：`uHandVel`，用于动量传递
- **速度衰减**：`multiplyScalar(max(0, 1 - dt * 8))`，手停住时拖拽归零
- **排斥强度**：`uHandActive * falloff * 1.4`
- **涡旋强度**：`uHandActive * falloff * 0.8`

### 相机系统
- **自动轨道**：0.12 rad/s 缓慢环绕
- **高度呼吸**：`1.5 + elevation * 3 + sin(t * 0.2) * 0.3`
- **注视点**：始终看向场景中心 (0, 0.5, 0)

## 六、形态切换过渡动画

### 切换流程
```
1. 触发切换 → uMorphFlash 从 0.6 柔和衰减至 0（1.4s，power2.out）
2. mixFactor 先降至 0.3（0.35s，power2.in）— 粒子短暂散开
3. mixFactor 回弹至原值（1.4s，power2.out）— 粒子重新聚合
4. targetPos 与 color 同步 GSAP 缓动过渡（2.2s / 1.8s，power3.inOut）
5. 随机延迟 0.3-0.9s，形成波浪式重组
```

### 视觉描述
> 旧形态如星尘般温柔解体，粒子向四周弥散，在空中悬停片刻；随后新形态的引力将它们重新牵引、归位，如生命体蜕变般优雅流畅。切换瞬间有一道柔和的淡蓝闪光掠过，转瞬即逝。

## 八、后处理管线（Post-Processing Pipeline）

### Pass 1: RenderPass
标准场景渲染输出

### Pass 2: UnrealBloomPass
| 参数 | 值 | 说明 |
|------|-----|------|
| Strength | 0.45 | 辉光强度（柔和）|
| Radius | 0.80 | 扩散半径（宽泛）|
| Threshold | 0.25 | 亮度阈值（仅高光参与）|

### Pass 3: 自定义 Finale Shader
```
径向色差（Chromatic Aberration）：
  RGB 通道边缘分离 0.0012，距中心越远偏移越大

暗角（Vignette）：
  smoothstep(0.95, 0.25, r² * 0.55)，边缘压暗至 0.55 倍

胶片颗粒（Film Grain）：
  仅边缘区域微弱噪点 0.02，中心干净

伽马校正：
  pow(col, 0.97) — 微降对比度，画面温润
```
